SFC生専用のSNS

## Description　
CNSアカウントを持つものだけが使用できるようなSNSがあればひとつのアカウントで（Keio.jpやsfsをいったりきたりせずに）多くの情報が得られるしツイッターに抵抗がある生徒も使いやすくなる。具体的には授業ごとにコメントできる場所、休講情報、課題期限のリマインドをする場所をもうけたりなどだ。
